Low-Code Studio

低代码开发平台

从这里进入工作流设计与大屏应用搭建，先选择项目入口，再进入现有编辑器继续开发。

创建工作流

直接进入当前工作流创建界面，开始拖拽节点和配置流程。

导入工作流

选择本地 JSON 文件，导入后自动进入当前工作流编辑界面。

创建大屏应用

进入独立的大屏应用编辑器，低代码搭建页面前端。

导入大屏应用

选择本地 JSON 文件并带入编辑器，便于继续完善大屏应用配置。

项目介绍

面向低代码开发的一体化实验平台

当前项目已经完成了工作流开发的后端能力，并具备可视化节点编排、执行和调试基础。 新增的主界面负责承接项目入口，让工作流与大屏应用都可以从统一门户进入。

工作流开发 节点拖拽、流程执行、单步调试
大屏应用扩展 独立大屏编辑器，支持组件拖拽、属性编辑和网页生成
JSON 导入 支持从主界面选择文件并带入编辑器继续编辑
统一入口 应用启动后先进入主页，再按项目类型进入编辑阶段
